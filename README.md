# F.I.N.V. - Lenguaje de Programación

F.I.N.V. es un lenguaje de programación creado para la materia "Sintaxis y Semántica de los Lenguajes"El lenguaje está inspirado en un "messi script", donde cada sentencia hace referencia a jugadas del futbolista argentino Lionel Messi.

## 🚀 Características
* **Intérprete propio:** Desarrollado e implementado mediante Pascal.
* **Componentes:** Incluye Analizador Léxico, Sintáctico y Semántico.
* **Tipado:** Los números se definen como reales.
* **Extensión:** El código fuente se ejecuta desde archivos `.txt`.

## 🛠️ Sintaxis y Palabras Reservadas

| Acción | Palabra Reservada |
| :--- | :--- |
| **Inicio de programa** | `Program [Nombre] pase`|
| **Fin de programa/bloque** | `gol`  |
| **Declaración de variables** | `var` |
| **Asignación** | `:=` |
| **Entrada (Input)** | `lio`  |
| **Salida (Print)** | `definir` |
| **Condicional (If)** | `asistir(condición) marcar`  |
| **Bucle (While)** | `gambetear(condición) marcar`  |

### Operadores Especiales
* **Potencia:** `zurda(base, exponente)`.
* **Raíz cuadrada:** `arco(radicando, índice)`.
* **Lógicos:** `and`, `or`, `not`.

## 🔧 Instalación
F.I.N.V. no requiere ningún tipo de instalación. Simplemente se debe descargar el paquete y ejecutar el archivo `finv.exe`.
